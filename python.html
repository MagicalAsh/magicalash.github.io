booleantools

A toolkit for working with boolean functions in an object-oriented way. Includes functions such as the Walsh transform, application of permutations, and correlation immunity. In addition, boolean tools includes tools for working with cellular automata and finite fields.

Upcoming features include vector operations and increased testing, along with the eventual release of version 1.0.

The code for the booleantools package is available on my Github. Feel free to report issues, add tests, and create pull requests.
